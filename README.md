## 🚀 Projects

### 🛒 Zenvora Living — End-to-End E-Commerce Testing
> *Comprehensive QA automation suite for a full-scale e-commerce platform*

Designed and executed a **complete end-to-end test strategy** for an e-commerce application, covering critical user journeys including product discovery, cart management, checkout workflows, and order lifecycle. Built reusable automation components using the **Page Object Model (POM)** architecture, significantly reducing test maintenance overhead.

- ✅ Automated **UI test cases** using Selenium WebDriver + TestNG with POM framework
- ✅ Validated **REST APIs** (product catalog, cart, orders) using REST Assured and Postman
- ✅ Executed **regression & smoke test suites** to ensure deployment stability
- ✅ Performed **database validation** using MySQL to verify data integrity end-to-end
- ✅ Simulated **concurrent user loads** with Apache JMeter for performance benchmarking
- ✅ Integrated test execution into **Jenkins CI/CD pipeline** for automated build triggers
- ✅ Generated detailed **TestNG HTML reports** for stakeholder visibility

### 🎪 Occasio — Multi-Role Event Management Testing
> *End-to-end BDD automation framework for a complex multi-tenant platform*

Led quality assurance for a **multi-role Event Management platform** serving organizers, attendees, and administrators. Implemented a **Behavior-Driven Development (BDD)** strategy using Cucumber, enabling clear communication between technical and non-technical stakeholders through human-readable test scenarios.

- ✅ Developed **Gherkin feature files** for business-critical workflows across Admin, Organizer & Attendee roles
- ✅ Built **step definitions** in Java with Cucumber BDD for cross-role scenario validation
- ✅ Performed **API contract testing** using Postman collections and automated environment switching
- ✅ Executed **role-based access control (RBAC)** validation across all user permission levels
- ✅ Validated **backend data consistency** via MySQL queries post every critical operation
- ✅ Benchmarked **event booking concurrency** under load using Apache JMeter
- ✅ Automated build and test triggers through **Jenkins pipeline integration**
