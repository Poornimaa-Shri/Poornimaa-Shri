### 🛒 Zenvora Living — End-to-End E-Commerce Testing

> *Comprehensive QA automation suite for a full-scale e-commerce platform*

Designed and executed a complete end-to-end test strategy covering critical user journeys — product discovery, cart management, checkout workflows, and order lifecycle — using a reusable **Page Object Model (POM)** architecture.

- ✅ Automated UI flows using **Selenium WebDriver + TestNG** with POM for low-maintenance test design
- ✅ Validated REST APIs — product catalog, cart operations, order management — using **REST Assured** and Postman
- ✅ Built regression & smoke test suites enabling rapid post-deployment confidence validation
- ✅ Verified backend data integrity via **MySQL** queries after every critical transaction
- ✅ Simulated concurrent user traffic with **Apache JMeter** for performance benchmarking
- ✅ Integrated test execution into **Jenkins CI/CD pipeline** for automated build-trigger runs

### 🎪 Occasio — Multi-Role Event Management Testing

> *BDD-driven automation for a complex multi-tenant event platform*

Led QA for a multi-role Event Management platform serving Admins, Organizers, and Attendees. Implemented a **Behavior-Driven Development (BDD)** strategy using Cucumber — enabling clear, human-readable test scenarios accessible to both technical and non-technical stakeholders.

- ✅ Authored **Gherkin feature files** covering business-critical workflows across all three user roles
- ✅ Built step definitions in Java with **Cucumber BDD** for cross-role end-to-end scenario automation
- ✅ Validated **Role-Based Access Control (RBAC)** across all user permission levels
- ✅ Performed **API contract testing** using Postman with automated environment switching
- ✅ Confirmed backend data consistency via **MySQL** queries post every booking and event operation
- ✅ Integrated automated test execution into **Jenkins** for continuous delivery validation
